# Rent Calculator 

## Description
Rent Calculator is a simple Python program that calculates the total monthly expenses
and splits the amount equally among all people living in a room or hostel.

It takes rent, grocery expenses, and electricity bill as input and shows how much
each person has to pay.

---

## Features
- Calculates total expense
- Splits amount per person
- Handles invalid inputs safely
- Easy to use for beginners

---

## Input Details
- Flat Rent
- Grocery Amount
- Electricity Bill
- Number of People

---

## Output
- Total Expense
- Amount to be paid by each person
